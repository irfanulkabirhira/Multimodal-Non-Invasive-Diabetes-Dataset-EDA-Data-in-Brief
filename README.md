# Multimodal-Non-Invasive-Diabetes-Dataset-EDA-Data-in-Brief
Multimodal dataset (15,720 sessions) for non-invasive diabetes screening — NIR optical sensor signals, wearable physiological data (heart rate, BVP, EDA), and demographics. Includes reproducible EDA, feature-tiering analysis, and a companion report. Licensed CC BY 4.0.
